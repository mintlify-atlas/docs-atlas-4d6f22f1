Fittura is a TypeScript-first React Native app built on the Expo platform, using Supabase for backend services and n8n for AI-powered automation workflows.

## Mobile framework

| Technology | Version | Purpose |
|---|---|---|
| Expo | ~54.0.32 | Cross-platform React Native toolchain |
| React Native | 0.81.5 | Mobile UI framework |
| expo-router | ~6.0.22 | File-based navigation |
| TypeScript | ~5.9.2 | Type safety |

## UI & styling

| Technology | Version | Purpose |
|---|---|---|
| NativeWind | ^4.2.1 | Tailwind CSS for React Native |
| Tailwind CSS | ^3.4.19 | Utility-first CSS framework |
| lucide-react-native | ^0.563.0 | Icon library |
| react-native-reanimated | ~4.1.1 | Animations |
| expo-haptics | ~15.0.8 | Haptic feedback |

## State management

| Technology | Version | Purpose |
|---|---|---|
| Zustand | ^5.0.10 | Global state (wardrobe + favorites) |
| React Context | — | Authentication state |

The wardrobe store holds items fetched from n8n and clears on logout. The favorites store is persisted locally via AsyncStorage using Zustand's `persist` middleware.

## Backend & storage

| Technology | Purpose |
|---|---|
| Supabase | Auth, user management, image storage |
| n8n | Automation workflows, AI classification, webhooks |

| Technology | Version | Purpose |
